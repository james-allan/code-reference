get_retries()

woocommerce-subscriptions/includes/payment-retry/data-stores/class-wcs-retry-hybrid-store.php : 123

Get a set of retries from the database

public get_retries([array<string|int, mixed> $args = array() ][, string $return = 'objects' ]) : array<string|int, mixed>
Parameters
$args : array<string|int, mixed> = array()

A set of filters: 'status': filter to only retries of a certain status, either 'pending', 'processing', 'failed' or 'complete'. Default: 'any', which will return all retries. 'date_query': array of dates to filter retries to those that occur 'after' or 'before' a certain date (or between those two dates). Should be a MySQL formated date/time string. 'orderby': Order by which property? 'order': Order in ASC/DESC. 'order_id': filter retries to those which belong to a certain order ID. 'limit': How many retries we want to get.

$return : string = 'objects'

Defines in which format return the entries. options: 'objects': Returns an array of WCS_Retry objects 'ids': Returns an array of ids.

since
2.4
