WooCommerce Subscriptions Add Cart Item.

A class to assist in the calculations required to add an item to an existing subscription. To enable proration, adding a product to a subscription inherits all the switch item (@see WCS_Switch_Cart_Item) functionality, however, doesn't have an existing item (@see WCS_Switch_Cart_Item::$existing_item) to replace.
