Constructor.

public __construct(string|callable $content, string $trigger[, string $content_type = 'plain-text' ][, string $heading = '' ][, array<string|int, mixed> $actions = array() ]) : mixed
Parameters
$content : string|callable

The content to display in the modal. This should be a string when $content_type is either 'plain-text' or 'html', a WooCommerce template filename when $content_type is 'template' or a function that echoes out the content when $content_type is 'callback'.

$trigger : string

A jQuery selector of the element which triggers the modal to be displayed.

$content_type : string = 'plain-text'

Optional. The modal content type. Can be 'plain-text', 'html', 'template' or 'callback'. Default is 'plain-text'.

$heading : string = ''

Optional. The modal heading text.

$actions : array<string|int, mixed> = array()

Optional. An array of actions to add to the modal. See {@see 'WCS_Modal::add_action'} for details on the action array format.
