public __construct([string $on_fatal = 'rollback' ][, string $on_shutdown = 'commit' ]) : mixed
Parameters
$on_fatal : string = 'rollback'

Optional. The type of query to run on fatal shutdown if this transaction is still active. Can be 'rollback' or 'commit'. Default is 'rollback'.

$on_shutdown : string = 'commit'

Optional. The type of query to run if a non-error shutdown occurs but there's still an active transaction. Can be 'rollback' or 'commit'. Default is 'commit'.

Closes out an active transaction depending on the type of shutdown.

public handle_shutdown() : mixed

Shutdowns caused by a fatal will be rolledback or commited @see $this->on_fatal. Shutdowns caused by a natural PHP termination (no error) will be rolledback or commited. @see $this->on_shutdown.
