woocommerce-subscriptions/vendor/subscriptions-core/includes/deprecated/class-wcs-filter-deprecator.php : 17

Handle deprecated filters.

When triggering a filter which has a deprecated equivalient from Subscriptions v1.n, check if the old filter had any callbacks attached to it, and if so, log a notice and trigger the old filter with a set of parameters in the deprecated format so that the current return value also has the old filters applied (whereever possible that is).
