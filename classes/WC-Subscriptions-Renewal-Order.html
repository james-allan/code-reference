Generate an order to record an automatic subscription payment.

public static generate_paid_renewal_order(int $user_id, string $subscription_key) : mixed

This function is hooked to the 'process_subscription_payment' which is fired when a payment gateway calls the @see WC_Subscriptions_Manager::process_subscription_payment() function. Because manual payments will also call this function, the function only generates a renewal order if the @see WC_Order::payment_complete() will be called for the renewal order.
