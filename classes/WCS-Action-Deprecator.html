woocommerce-subscriptions/vendor/subscriptions-core/includes/deprecated/class-wcs-action-deprecator.php : 16

Handle deprecated actions.

When triggering an action which has a deprecated equivalient from Subscriptions v1.n, check if the old action had any callbacks attached to it, and if so, log a notice and trigger the old action with a set of parameters in the deprecated format.
